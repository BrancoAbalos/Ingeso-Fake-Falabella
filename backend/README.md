# Backend (placeholder)

Carpeta dedicada a la lógica del backend. Actualmente solo contiene archivos base para iniciar un servicio en Node.js/Express o cualquier otra tecnología.

Siguientes pasos sugeridos:
- Elegir framework (Express, Fastify, NestJS, etc.)
- Añadir `package.json`, endpoints básicos (productos, carrito)
- Añadir `Dockerfile` y `.dockerignore` para contenerizar el servicio
- Crear `docker-compose.yml` en la raíz si quieres orquestar frontend+backend

Ejemplo de comandos (desde `backend/`):

```powershell
npm init -y
npm install express
```
